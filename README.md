Customer Behavior Analysis
Overview
An end-to-end data analytics project analyzing customer behavior and purchase patterns. Built using Python (Pandas, NumPy), SQL (PostgreSQL), and Power BI, this project demonstrates the complete workflow from raw data ingestion to interactive dashboard reporting.

Workflow
Data Loading & Cleaning
Imported dataset into Python.
Performed EDA and cleaned data using Pandas.

SQL Queries
Ran aggregations, joins, and segmentation queries in PostgreSQL.

Dashboard Development
Designed interactive Power BI dashboard with filters for gender, subscription status, season, and shipping type.

Reporting & Presentation
Compiled findings into a structured report and presentation.

📊 Key Insights
Customer Base: 654 customers, generating $38.23K revenue.
Subscription Gap: 73% are non-subscribers → opportunity to grow subscriptions.
Product Drivers: Clothing dominates revenue; fashion items (skirts, jeans, sneakers) are top contributors.
Age Groups: Balanced revenue contribution across demographics.
Customer Satisfaction: Average review rating of 3.82, suggesting room for improvement.
